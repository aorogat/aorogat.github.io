Research Overview

My research investigates foundational problems in multi-agent coordination and memory-augmented reasoning over large-scale knowledge graphs. I am particularly interested in how collections of specialized agents can decompose complex reasoning tasks, maintain persistent memory structures, and coordinate effectively to produce verifiable, interpretable results.

During my doctoral work at Carleton University under Professor Ahmed El-Roby, I developed large-scale benchmarking frameworks and automated evaluation platforms for knowledge graph question answering. This work addressed fundamental challenges in dataset construction, evaluation bias, and system scalability—culminating in contributions to SIGMOD, VLDB, and CIKM.

I currently work with Professor Essam Mansour at Concordia University, where my research explores how large language models can be orchestrated into reliable multi-agent systems capable of structured reasoning, tool execution, and long-horizon planning. This work builds on prior experience developing production knowledge graph systems at Ericsson for CloudRAN analytics.

Research Agenda

Multi-Agent Architectures for Knowledge-Grounded Reasoning

Core question: How can we design agent frameworks where specialized reasoning modules coordinate effectively over structured knowledge sources? My work addresses this through the development of agent orchestration platforms that decompose complex queries into retrieval, planning, and validation subtasks, with each agent maintaining appropriate memory and communication protocols.

Benchmark Design and Systematic Evaluation

Core question: What evaluation methodologies enable fair, comprehensive assessment of knowledge-intensive AI systems? I have developed automated benchmark generation platforms and large-scale annotated datasets to address coverage limitations and evaluation bias in knowledge graph question answering.

Memory and Persistence in Agent Systems

Core question: How should agents maintain, update, and reason over persistent memory structures? This research direction examines adaptive memory models, reflective planning mechanisms, and techniques for long-horizon reasoning that require agents to accumulate and synthesize information across extended interactions.

Principal Research Contributions

Chatty-KG: Multi-Agent Question Answering over Knowledge Graphs

SIGMOD 2026 · Multi-Agent Systems

Developed a multi-agent architecture for knowledge graph question answering using LangGraph-based orchestration. The system decomposes KGQA into specialized reasoning roles—entity retrieval, query planning, and answer validation—enabling robust multi-step reasoning over large knowledge graphs. This work demonstrates how agent specialization and structured communication protocols can improve both accuracy and interpretability in complex question answering tasks.

QueryBridge: A Million-Scale Dataset for Knowledge Graph Question Answering

CIKM 2025 · Dataset Release

Designed and released a large-scale dataset containing over one million natural language questions paired with executable SPARQL queries. QueryBridge addresses fundamental data scarcity problems in natural language to SPARQL translation research, providing diverse query patterns across multiple knowledge graphs. The dataset enables systematic training and evaluation of semantic parsing systems at scale.

Automated Benchmark Generation for Knowledge Graph QA

SIGMOD 2023, VLDB 2022 · System: Maestro

Created an automatic benchmark generation framework addressing evaluation bias and coverage limitations in knowledge graph question answering. The system generates diverse, difficulty-controlled question sets through template-based synthesis and constraint solving, enabling more comprehensive and fair assessment of KGQA systems across multiple dimensions of linguistic and structural complexity.

Ericsogate: Enterprise Knowledge Graph Platform

CIKM 2024 · Industry Collaboration with Ericsson

Architected and deployed a production-grade knowledge graph platform for CloudRAN analytics at Ericsson. The system implements a six-layer architecture covering data ingestion, semantic modeling, graph construction, and application services. It provides semantic search, automated summarization, and retrieval-augmented exploration over heterogeneous operational data, demonstrating the viability of knowledge graph abstractions in large-scale industrial settings.
