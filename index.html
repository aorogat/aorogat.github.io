About Me

Abdelghny Orogat is a PhD candidate at Carleton University, specializing in Question Answering over Knowledge Graphs. His research has led to several publications in prestigious journals and conferences (e.g., SIGMOD, VLDB, and CIKM). Abdelghany's industry experience encompasses knowledge graph construction, management, summarization, and developing applications that leverage these graphs. His proficiency in Natural Language Processing (NLP) has significantly enhanced his contributions to the field. In addition to his research endeavors, Abdelghany has held academic positions as a Contract Instructor (Lecturer) at Carleton University. He also holds a Master of Science in Computer Engineering from Tanta University, Egypt, where his thesis focused on adapting webpages to convert unresponsive designs into responsive formats.

Selected Projects

Star: A Question Answering System over Knowledge Graphs [Carleton University (In Progress)]

Description: Developed a question-answering system designed to interpret natural language queries using large language models (LLMs). The system leverages techniques like Retrieval-Augmented Generation (RAG) and LoRA fine-tuning to enhance accuracy and efficiency in retrieving and answering queries over complex knowledge graphs. One stage of the project focuses on defining natural language query (NLQ) components, utilizing LLaMA-3 fine-tuned on the QueryBridge dataset, as demonstrated in the video.

Technologies: PyTorch, LlamaIndex, OpenAI API, Semantic Web

QueryBridge: One Million Annotated Questions with SPARQL Queries - Dataset for Question Answering over Knowledge Graph [Carleton University (2024)]

Description: Created an extensive dataset containing over one million annotated questions with SPARQL queries. QueryBridge is aimed at supporting research in question-answering systems by providing a high-quality benchmark for natural language queries over knowledge graphs.

Technologies: Java, Apache Jena, RDF, SPARQL, Semantic Web

Improved RAG: Two-Stage Retrieval-Augmented Generation System [Carleton University (Completed)]

Description: Developed a two-stage approach to improve the Retrieval-Augmented Generation (RAG) system for large language models. Our approach combines the efficiency of a single embedding for fast retrieval with the accuracy of a multi-embedding strategy, leveraging the COLBERT model. In the first stage, we quickly retrieve a set of the top documents from a large collection using single embeddings. Then, in the second stage, we apply multi-embedding retrieval on this subset to identify the best match. For this project, we used Milvus as the vector database to manage embeddings effectively.

Technologies: PyTorch, COLBERT Model, Milvus, Vector Databases

Ericsogate: Knowledge Graph System for CloudRAN Data Management [Ericsson (2023)]

Description: Developed a six-layer knowledge graph framework to efficiently manage CloudRAN data. The system enhances data accessibility, adaptability, and insight generation for telecom applications by using a modular and scalable architecture. The video on the left shows the architecture of Ericsogate. Please note that the code is not publicly available.

Technologies: Java, RDF, OWL, Apache Jena, Python

Knowledge Graph Summarizer [Ericsson (2023)]

Description: Created a summarization tool for knowledge graphs, designed to simplify large datasets by identifying key patterns and aggregating related nodes and edges. This project improves data analysis and allows stakeholders to quickly understand trends in complex datasets.

Technologies: Java, Semantic Web, RDF, OWL, Apache Jena

Maestro: Benchmark Generation System [Carleton University (2022)]

Description: Designed a system for automatic benchmark generation tailored for question-answering systems over knowledge graphs. Maestro generates benchmarks that cover linguistic and syntactic properties, providing a thorough testbed for evaluating QA systems.

Technologies: Java, Apache Jena, Python, NLTK
