About Me

Abdelghny Orogat holds a PhD from Carleton University, where his groundbreaking research focused on advancing Question Answering over Knowledge Graphs (QAKG). During his PhD, he fine-tuned Large Language Models (LLMs) using his curated one-million-question dataset, significantly enhancing QAKG capabilities. He introduced Maestro, a system for natural language query (NLQ) generation from knowledge graphs, published in prestigious venues like SIGMOD and VLDB. Abdelghny also developed Ericsogate, a framework for constructing knowledge graphs from diverse data sources, revolutionizing data integration and management. His extensive industry experience spans knowledge graph construction, management, summarization, and application development. Alongside his research, he served as a Contract Instructor (Lecturer) at Carleton University. Abdelghny also holds a Master of Science in Computer Engineering from Tanta University, Egypt, where his thesis focused on adapting webpages to convert unresponsive designs into responsive formats.

Selected Projects

[Carleton University (In Progress)] Star: A Question Answering System over Knowledge Graphs

Description: Developed a question-answering system designed to interpret natural language queries using large language models (LLMs). The system leverages techniques like Retrieval-Augmented Generation (RAG) and LoRA fine-tuning to enhance accuracy and efficiency in retrieving and answering queries over complex knowledge graphs. One stage of the project focuses on defining natural language query (NLQ) components, utilizing LLaMA-3 fine-tuned on the QueryBridge dataset, as demonstrated in the video.

Technologies and Approaches: PyTorch, Torchtune, LLM (Fintuning, RAG, Ollama, LlamaIndex, OpenAI API), Semantic Web

[Carleton University (2024)] QueryBridge: One Million Annotated Questions with SPARQL Queries - Dataset for Question Answering over Knowledge Graph

Description: Created an extensive dataset containing over one million annotated questions with SPARQL queries. QueryBridge is aimed at supporting research in question-answering systems by providing a high-quality benchmark for natural language queries over knowledge graphs.

Technologies: Java, Apache Jena, RDF, SPARQL, Semantic Web, Python, NLTK

[Ericsson (2024)] Improved RAG: Two-Stage Retrieval-Augmented Generation System

Description: Developed a two-stage approach to improve the Retrieval-Augmented Generation (RAG) system for large language models. Our approach combines the efficiency of a single embedding for fast retrieval with the accuracy of a multi-embedding strategy, leveraging the ColBERT model. In the first stage, we quickly retrieve a set of the top documents from a large collection using single embeddings. Then, in the second stage, we apply multi-embedding retrieval on this subset to identify the best match. For this project, we used Milvus as the vector database to manage embeddings effectively.

Technologies: PyTorch, COLBERT Model, Milvus, Vector Databases

[Ericsson (2023)] Ericsogate: Knowledge Graph System for CloudRAN Data Management

Description: Developed a six-layer knowledge graph framework to efficiently manage CloudRAN data. The system enhances data accessibility, adaptability, and insight generation for telecom applications by using a modular and scalable architecture. The video on the left shows the architecture of Ericsogate. Please note that the code is not publicly available.

Technologies: Java, RDF, OWL, Apache Jena, Python

[Carleton University (2021)] CBench: Towards Better Evaluation of Question Answering Over Knowledge Graphs

Description: CBench is an extensible benchmarking suite for evaluating question-answering systems over knowledge graphs. It provides comprehensive evaluation metrics and customizable test scenarios, enabling researchers and practitioners to assess the accuracy and performance of QA systems on structured data. The demo video on the left illustrates its capabilities and usage in various QA scenarios.

Technologies: Java, JavaServer Faces (JSF), SPARQL, RDF, Apache Jena, HTML, CSS, Java Script
